# Tourism-Dashboard
I have created a dashboard based on India’s Domestic Tourism Analysis.

**Short Descrition**
Domestic tourism is tourism involving residents of one country traveling only within that country. Such a vacation is known as a domestic vacation. The Ministry of Tourism compiles data on domestic visits in States/UTs based on the information received from the State Government/UT Administrations. Such information is collected by their Tourism Departments.

This dashboard contains 9 years of data i.e. from year 2014 to year 2023.
Here are my Key findings:
1)	Domestic tourism in India expanded until 2019, peaked, then dropped sharply during Covid(2020-2021) and recovered significantly by 2023.
2)	Top Performing States: Uttar Pradesh (Share of total 2023-21%)
                           Tamil Nadu (Share of total 2023-12%)
                           Karnataka (Share of total 2023-12%)
  	
Anomalies:
1)	Incomplete “Others” category
2)	Missing states in later years like Jharkhand data missing after 2014

**Tech Stack**
The Dashboard was built using following tools and Techniques:
1. Power Bi Desktop - Main data visualization platform used for report creation.
2. Power Query - DAX transformation and cleaning layer for reshaping and preparing the data.
3. DAX - Used for calculated measures, dynamic visuals and conditional logic.
4. File Format - .pbit for development and .png for dashboard previews.

**Data Source**
Downloaded dataset from ndap.niti.gov.in
